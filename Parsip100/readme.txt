Development Machine:
1. CPU: Intel Quad Core Processor @2.66 GHz
2. GPU: NVIDIA GeForce 8800 Ultra
3. RAM: 4GB

Dependencies and external libraries:

1. Qt library from Nokia V4.7.0. Available under LGPL license at: http://qt.nokia.com/
2. DirectX 11. June 2010 SDK. Available at: http://www.microsoft.com/downloads/en/details.aspx?displaylang=en&FamilyID=3021d52b-514e-41d3-ad02-438a3ba730ba
3. NVidia Cuda GPU computing SDK V3.2 RC. Available at: http://developer.nvidia.com/object/cuda_3_2_toolkit_rc.html
4. Intel Threading Building Blocks TBB available for free from: http://www.threadingbuildingblocks.org/
